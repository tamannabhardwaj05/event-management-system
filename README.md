# event-management-system
A complete Event Management Web Application developed using PHP & MySQL.
This project allows students to register for events and enables admin to manage events, registrations, and important notices.

# Features :
* User Side -
-View upcoming events
-Register for events
-Explore event gallery
-View important notices
-Responsive and clean UI
* Admin Panel-
-Secure admin login system
-Add / Edit / Delete events
-View all student registrations
-Delete registrations
-Add important notices (visible on homepage)
-Live dashboard (total events & registrations)

# Technologies Used
Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL
Server: XAMPP

# Setup Instructions
1. Install XAMPP
2. Start Apache and MySQL
3. Copy project folder to: C:/xampp/htdocs/
4. Open browser and go to: http://localhost/phpmyadmin
5. Create a database (e.g., event_db)
6. Import the SQL file (tables: events, registrations, notices, admin)

# Run project: http://localhost/jims/
